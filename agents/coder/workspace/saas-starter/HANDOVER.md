## üåç Deployment Checklist
To go to production (Vercel/Netlify):

1.  **Database:**
    - Spin up a PostgreSQL database (Neon, Supabase, Railway).
    - Update `.env`: `DATABASE_URL="postgres://..."`
    - Update `prisma/schema.prisma`: `provider = "postgresql"`
    - Run `bun prisma db push`.

2.  **Authentication:**
    - Update `.env`: Add `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`.
    - Edit `src/lib/auth.ts`: Comment out `CredentialsProvider`, uncomment `GoogleProvider`.

3.  **Payments:**
    - Update `.env`: Add `STRIPE_SECRET_KEY`, `STRIPE_WEBHOOK_SECRET`.
    - Create Stripe Webhook pointing to `https://your-domain.com/api/webhooks/stripe`.

4.  **Deploy:**
    - Push to GitHub.
    - Connect repo to Vercel.
    - Add Environment Variables in Vercel Dashboard.
